    \index{Drains}
    \begin{verbatim}
    Pre-requisites:
    Activate one of \gwf,\swf\ or\cln\ domains
    Choose cells

    Instructions:
    gwf drain

    Inputs:
    Drain conductance  L/T
    \end{verbatim}

    All chosen cells will be assigned a drain elevation equal to the top elevation of the cell with the specified drain conductance.
